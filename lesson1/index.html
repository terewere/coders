What is Web Development?

Web development is the process of building websites and applications for the internet.

Web developers are responsible for designing and developing websites.

Web development can be group into 2 major categories: frontend development and backend development

Frontend Development

Frontend development refers to everything that the end user experiences directly. Languages for frontend development includes HTML, CSS, JavaScript etc.
